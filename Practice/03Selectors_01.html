selectors

CSS selectors are patterns used to select and style HTML elements. Here are some basic types of selectors:

1. Universal Selector (*): Selects all elements.

* { margin: 0; padding: 0; }

2. Element Selector (element): Selects all elements of a specific type.

p { font-size: 16px; }

3. Class Selector (.class): Selects all elements with the specified class.

.highlight { background-color: yellow; }

4. ID Selector (#id): Selects an element with the specified ID.

#header { font-size: 24px; }

5. Attribute Selector ([attribute=value]): Selects elements with the specified attribute and value.

[type="text"] { border: 1px solid #000; }

6. Descendant Selector (ancestor descendant): Selects all descendant elements of a specified ancestor.

div p { color: blue; }

7. Child Selector (parent > child): Selects all direct children of a specified parent.

div > p { font-weight: bold; }

8. Adjacent Sibling Selector (previous + next): Selects the next sibling element immediately following the specified element.

h1 + p { color: green; }

9. General Sibling Selector (previous ~ siblings): Selects all sibling elements following the specified element.

h1 ~ p { text-decoration: underline; }

10. Pseudo-class Selector (:pseudo-class): Selects elements in a specific state.

a:hover { color: red; }

11. Pseudo-element Selector (::pseudo-element): Selects and styles parts of an element.

p::first-line { font-weight: bold; }
